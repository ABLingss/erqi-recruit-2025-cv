## 人工神经网络实现手写数字数据集(MNIST)识别

本题目为2025年电子科技大学尔绮工作室机器学习招新题

## 使用方法
1. 克隆本仓库
```bash
git clone https://github.com/ABLingss/erqi-recruit-2025-cv.git
```
2. 安装依赖
```bash 
pip install -r requirements.txt
```
3. 运行程序
